#Exercício para a aula de 20/10/2016

##Para manter uma lista de objetos ordenada, é necessário que seja possível comparar objetos para identificar qual dos objetos deve ser posicionado antes do outro.

## Esta comparação para tipos de dados fundamentais pode ser realizada com os operadores de comparação (<, >, <=, >=, ==), no entanto, no caso de tipos de dados referenciados (p.ex.: arrays e objetos), esta comparação não pode ser realizada a partir destes operadores, porque a comparação será realizada através do endereço da referência e não a partir dos dados dos objetos.

##O objetivo deste exercício é estudar e implementar as duas formas como esta comparação entre objetos pode ser realizada em Java.

###Explique o funcionamento dessas duas formas de comparação de objetos, e mostre o código necessário para comparar uma classe "Pessoa", com  um atributo "nome", cujos objetos devem ser comparados.
